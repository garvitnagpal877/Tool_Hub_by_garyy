Ageing Analysis Tool

Calculate ageing days, classify records into ageing buckets, generate ageing summaries, and export the results into Excel.

Purpose

The Ageing Analysis Tool is used to calculate ageing days, classify records into ageing buckets, generate ageing summaries, and export the results into Excel.

Step 1 – Upload File

  1. Open the Ageing Analysis Tool.
  2. Click "Drop Excel File Here" or drag and drop an Excel file.
  3. Supported formats:
    • .xlsx
    • .xls
    • .xlsm
    • .csv

After upload, the system will display:

  • Available sheets
  • Row count
  • Column count
  • Data preview

Step 2 – Configure Data

Source Sheet

Select the sheet containing the data.

Data Starts at Row

Enter the row number where actual data begins.

Example:

  • Header in Row 1 → Enter 2
  • Header in Row 3 → Enter 4

Step 3 – Select Ageing Method

Method 1 – Start Date vs End Date

Used when both dates are available.

Example: Invoice Date, Payment Date

Ageing = End Date – Start Date

Method 2 – Date vs Fixed Date

Used when ageing is to be calculated till a specific date.

Example: Invoice Date, 31-Mar-2026

Ageing = Fixed Date – Invoice Date

Method 3 – Date vs Today

Used for current outstanding ageing.

Ageing = Today – Invoice Date

Step 4 – On-Time / Grace Period

No Grace Period

Ageing starts immediately.

Fixed Days

Example: 30 Credit Days

Ageing starts after 30 days.

From Column

Uses payment terms available in the data.

Example: Credit Days Column

Step 5 – Define Age Buckets

Enter bucket ranges:

0-30, 31-60, 61-90, 91-180, 181-999999

The tool automatically classifies each record into the applicable bucket.

Bucket Range (Days)
Bucket 1 0 – 30
Bucket 2 31 – 60
Bucket 3 61 – 90
Bucket 4 91 – 180
Bucket 5 181 and above

Step 6 – Summary Rules

Select calculations to be generated.

Available Functions:

  • Count
  • Sum
  • Average
  • Minimum
  • Maximum

Examples:

  • Count of Invoice Number
  • Sum of Amount
  • Average Ageing

Step 7 – Run Analysis

Click: "Run Ageing Analysis"

The tool will:

  1. Calculate ageing days.
  2. Apply grace period.
  3. Assign ageing bucket.
  4. Generate summary.
  5. Display results.

Step 8 – Review Results

Outputs generated:

Calculation Summary

Shows:

  • Total Records
  • Average Ageing
  • Maximum Ageing
  • Bucket Distribution

Ageing Data Preview

Displays processed records.

Ageing Summary

Shows bucket-wise analysis.

Step 9 – Export Results

Click: "Download Ageing Analysis Excel"

Output Excel contains:

  • Detailed Ageing Sheet
  • Ageing Summary Sheet
